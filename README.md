# so-tech-test

## Requirements
* Ruby >= 3.1.x

## Installation
    $ bundle install
    $ rails db:migrate

## Usage
    $ rails s
    
### Endpoints
- Categories:  
http://localhost:3000/categories


- Recipes for a specific category ID:  
http://localhost:3000/categories/1/recipes


- Recipe:  
http://localhost:3000/recipes/52874

## Running tests
    $ bundle exec rails test
